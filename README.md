## video-controller

### 说明
1. 本插件现在仅支持腾讯视频，爱奇艺，优酷视频,555dy 跳过片头】片尾和跳过中间某一段时间。
2. 暂时只支持以下域名:
    - v.qq.com
    - www.bilibili.com
    - www.iqiyi.com
    - v.youku.com
    - 555pian.com
3. 本插件现在只是第一版，可能会有bug, 如有bug或者更好的建议，请提问到issues(https://github.com/linuxkai/video-controller/issues)(请标注使用系统型号和chrome的版本)
